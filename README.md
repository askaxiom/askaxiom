## About Axiom AI

Axiom AI is an intelligent decision platform designed to reduce research time and increase confidence before making important decisions.

Instead of giving endless links or generic answers, Axiom investigates available information, compares alternatives and provides clear, evidence-based recommendations.

### Core Principles

- Research before recommendation
- Evidence over opinions
- Explain every recommendation
- Be transparent when confidence is low
- Save the user's time
